Ride Share

In this project, I applied Ruby object oriented design to create a ride share program. This RideShare module consisted of four classes. Hotel

Hotel is a program which allows a hotel administrator manage all reservations of a hotel containing 20 rooms. I designed this program using Ruby. The program allows the admnistrator to view a list of rooms available on a specific date range, return all reservations for a single date, make reservations and create blocks.

Word Guess Game

Word Guess is a fun game that I created with my pair programming partner Phoebe. This was my first time working on a pair programming project. I enjoyed the experience as I learned a lot from Phoebe's coding style.

The theme of the game is The Simpsons. Users are provided with a random word (selected from Faker) to guess. The player has up to 5 tries to guess the full word.
