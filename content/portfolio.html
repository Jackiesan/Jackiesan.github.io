Word Guess Game

Word Guess is a fun game that I created with my pair programming partner Phoebe. This was my first time working on a pair programming project. I enjoyed the experience as I learned a lot from Phoebe's coding style.

The theme of the game is The Simpsons. Users are provided with a random word (selected from Faker) to guess. The player has up to 5 tries to guess the full word.
